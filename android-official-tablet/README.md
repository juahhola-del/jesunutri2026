# JESUnutri Android Oficial Tablet

App Android oficial para la tablet de Jesu.

Esta app no reemplaza todavia `android-capturer`. La tablet oficial contiene:

- backend local Android en `127.0.0.1:8787`;
- SQLite local privado de la app;
- interfaz web local empacada dentro del APK;
- importacion inicial desde Supabase mediante login admin;
- capturador integrado para aprender etiquetas e ingresar productos;
- servicio Android en primer plano para recibir capturas desde otros dispositivos.

El flujo operativo diario no debe depender de Vercel, PC ni Node externo.
